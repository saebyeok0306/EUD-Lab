    베이스

    Base

    SCM Draft2

    SCM Draft 2는 현재 가장 많이 쓰이고 있는 맵 에디터입니다. 이전에 나왔던 X-tra, 스타포지의 기능 모두를 가지고 있습니다. 해당 에디터는 가장 기본적인 에디터로, EUD와 관련된 기술은 별도의 EUD Editor를 활용하는게 좋습니다.

    개인적으로는 최신버전은 버그가 많아서 2019년도 버전을 쓰시는 걸 추천합니다.

    링크

    TrigEditPlus v2.0

    PPS님이 개발하신 툴로, Lua를 지원하지 않는 특징이 있습니다.

    링크

    TrigEditPlus v3.0

    Ninfia님이 개발하신 툴로, 최신 SCM Draft2도 지원하도록 패치된 버전입니다.

    링크

    EUD

    EUD

    EUD Editor2

    맛있는빙수님이 만드신 에디터입니다. 현재 가장 최신버전은 SE 0.18.1.7입니다. StarEdit.net 유저들과 아르타니스님이 이어서 개발한 버전으로 버그 수정, 에디터 성능 개선, UI 조절, 영어 번역 개선 등 여러 방면으로 EE2를 개선한 버전입니다.

    링크

    EUD Editor3

    편의 기능이 개선되고, SCA 유즈맵을 개발하기 위해서는 반드시 필요한 맛있는빙수님의 에디터입니다.

    링크 패치노트

    EUD Draft

    트리거왕님께서 개발 및 현재는 아르타니스님께서 이어서 개발하고 있는 툴입니다. Eudplib, epScript을 활용해서 작성한 코드파일을 유즈맵으로 컴파일하기 위해 필요합니다.

    링크 패치노트

    VS Code (Visual Studio Code) IDE

    오픈소스 코드 에디터입니다. EUD Editor3 환경에서 Eudplib, epScript와 같이 프로그래밍 언어를 활용해서 개발할 경우 필요합니다. 확장프로그램(Extensions)에서 함주한님의 eps-server를 별도로 설치해서 쓰셔야 합니다.

    링크

    기타

    기타

    SC1 Manager

    고래밥은맛있어님이 개발한 툴로, 스타크래프트1 다중실행, 종료를 지원합니다. 혼자서 멀티플레이 테스트를 하고자 할때 도움이 됩니다.

    링크

    SCPM (Starcraft Picture Mapper)

    이미지 파일을 맵파일(.scm, .scx)로 변환해주는 툴입니다.

    ！사용시 주의사항
    SCPM5 기준, 'Place start locations', 'Place map revealers' 옵션을 체크해제하고 'Create uncompressed scenario'를 체크한 뒤에 생성해야 합니다. 또한 SCPM으로 생성된 맵파일을 베이스로 작업하시면 안되고, SCM Draft2에서 직접 새맵을 생성한 뒤 지형을 복사 붙여넣기하여 써야 합니다.

    링크

    CS_Minimap (썸네일 편집기)

    Ninfia님이 개발하신 툴로, 지형이 아닌 스타트로케이션이 미니맵 프리뷰에 보인다는 점을 이용하여 미니맵에 이미지를 삽입하는 툴입니다.

    링크

    CS_FogofWar (전장의 안개 편집기)

    Ninfia님이 개발하신 툴로, 전장의 안개 (Fog of War)를 삽입한 이미지로 만드는 툴입니다.

    링크

    CS_Tileset (지형 편집기)

    Ninfia님이 개발하신 툴로, 지형에 다양한 패턴을 적용합니다.

    링크

    CS_STRConverter (문자열 변환기)

    Ninfia님이 개발하신 툴로, strx str, CP949 UTF-8 변환을 지원합니다.
